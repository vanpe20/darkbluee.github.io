# Travel photos

Place travel photographs in this directory. Use lowercase, descriptive filenames such as:

- `san-francisco-01.jpg`
- `yosemite-sunset-01.jpg`

Then add the place and its photo list to `travel-data.js`. The latitude and longitude place its marker on the map automatically. A marker is shown only when at least one photo is present.

The world map is adapted from `BlankMap-Equirectangular.svg` on Wikimedia Commons, created from Natural Earth data and released under CC0.
